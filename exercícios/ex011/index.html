Imagens Dinamica

O conceito é que as imagens vão se adaptando ao tamanho, se passar de um determinado tamanho ela troca para outra imagem.
Existem 3 tipos de imagens nesse site:

  • Pequena
  • Média
  • Grande

Aúdios no site

Em geral, aquivos mp3 são pesados e acabam prejudicando o site.Existem 3 formatos de arquivos de áudio:

  1. mpeg - MP3
  2. WAV - péssimo - pesado demais
  3. OGG
